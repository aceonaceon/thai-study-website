## 一、專案目標

### 核心定位
成為**繁體中文世界最權威的泰國留學資訊網站**，透過語意矩陣內容策略，建立 SEO 流量漏斗，最終將潛在客戶引流至學無界代辦服務。

### 商業模式
```
免費內容（SEO 流量）→ 建立信任 → CTA 引流 → 學無界代辦服務 → 轉換
```

## 二、技術架構

### 2.1 架構圖
```
┌─────────────┐     Webhook      ┌──────────────┐
│   Notion    │ ────────────────→│   GitHub     │
│  (CMS/DB)   │                  │  (Source)    │
└─────────────┘                  └──────┬───────┘
                                        │
                                   GitHub Action
                                        │
                                        ▼
                                 ┌──────────────┐
                                 │  Cloudflare  │
                                 │    Pages     │
                                 └──────────────┘
```

### 2.2 技術棧
- **CMS**：Notion Database
- **前端框架**：Next.js 16（App Router）+ React 19
- **部署**：Cloudflare Pages（靜態生成）
- **樣式**：Tailwind CSS
- **圖片**：Cloudflare R2 / FAL AI 生成
- **CI/CD**：GitHub Actions + Notion Webhook

### 2.3 自動部署流程
1. Notion 新增/編輯文章 → Webhook 觸發
2. GitHub Action 啟動 → 從 Notion 拉取內容
3. Next.js Build → 生成靜態頁面
4. Cloudflare Pages Deploy → 網站更新

## 五、前端設計（完整版 by Samantha）

> 🎨 本節由 Samantha 擴充，包含完整配色系統、版面設計、圖片規劃

### 5.1 設計理念

**品牌調性**
- **專業可信**：留學代辦需要建立信任感
- **溫暖親切**：像顧問在跟學生聊天
- **現代感**：符合年輕族群審美
- **泰國元素**：融入熱帶、陽光、友善的氛圍

**設計原則**
1. **簡潔為王**：資訊架構清晰，不雜亂
2. **圖片說話**：大量使用高品質圖片
3. **留白呼吸**：充足的白空間
4. **CTA 突出**：行動呼籲明顯但不突兀
