# Director's Treatment — Thai Study Website

## Director Brief
- Visual thesis: A structured journey through warmly lit compartments, each revealing a facet of Thai education like rooms on Wes Anderson's Darjeeling Limited train
- Signature technique 1: Centered symmetry + planimetric framing → center-aligned layouts, symmetrical grids, content framed within decorative borders
- Signature technique 2: Lateral tracking / compartment cross-sections → horizontal scroll journey for process sections, bento grid for category exploration
- Signature technique 3: Curated clutter within ornamental frames → dense but organized information within warm, framed containers
- Motion rules: Slow eased transitions (600ms+), lateral movement preferred, max 1 heavy interaction per page, max 2 showy reveals. No bouncy/spring animations.
- Typography rules: Fraunces (heading serif) + Work Sans (body sans) + Noto Serif TC / Noto Sans TC for Chinese. Centered headings with generous letter-spacing.

## Site Cinematic Grammar
- Page-shell logic: Warm cream canvas (#F5ECD7) with content in framed compartments. Each section is a "room" with subtle borders or background shifts.
- Navigation posture: Centered logo with symmetrical nav links flanking it. Thin top border accent in brass gold. Understated, precise.
- Framing discipline: Content lives within implied frames — subtle borders, background color shifts, or ornamental line separators. Never floating in open space.
- Density cadence: Alternating between dense compartment grids and breathing-room sections. No two adjacent sections at the same density.
- Recurring material layers: Warm paper texture at low opacity, brass/gold line separators, subtle grain on hero images
- Allowed composition families: Centered stack, Bento grid, Z-split alternating, Horizontal overflow, Card within card
- What may repeat: Centered alignment, warm shadow treatment, ornamental line separators
- What must vary page to page: Hero composition, section rhythm, primary interaction pattern
- Demo uniqueness guardrail: No gradient overlays, no generic card grids, no SaaS section cadence

## Page Arc — Home

- Page-role scene: The train station — arriving at the beginning of a journey
- Page scene thesis: The homepage is a grand departure hall where every destination is visible through ornamental windows
- One big idea: Compartmented symmetry — content viewed through framed openings like train windows
- Hero dominance statement: A centered monumental title over a warm, film-graded photograph with ornamental border framing — not a gradient, but a directed composition
- Restraint statement: No parallax, no gradient overlays, no floating badges. Let the framing and typography carry authority.
- Material thesis: Warm cream paper, brass rule lines, aged-photography color grading on images
- Typography thesis: Large centered Fraunces headings with generous letter-spacing create chapter-title authority
- Narrative arc: Wes Anderson Default — B4 → B2 → B6 → B7 → B15 → B16 → B19 → B20 → B22
- Hero archetype: Chapter Zero prologue merging into Atmosphere Bath
- Signature composition: Bento quadrant — 4 unequal category windows arranged as a cross-section view
- Grid fallback test: If reduced to a 4-column equal grid, the compartment framing and size hierarchy would be lost
- Shared system holdback: Navigation and footer derive after page compositions lock
